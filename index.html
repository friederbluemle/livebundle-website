LiveBundle

For React Native

Get Started

Share application changes

Swiftly and conveniently share application changes during development. LiveBundle generates a QR Code that can just be scanned from the application, or a deeplink that can be navigated to.

GitHub Integration

LiveBundle comes with a GitHub integration.
When configured, LiveBundle will automatically post a QR Code and deep link on every pull request. Neat !

100% Open Source

LiveBundle is fully open sourced.
We provide the plumbing, you'll just need to provide the storage.
